**Krishi Mitra AI** is a cutting-edge, data-driven intelligence layer for the Indian agricultural ecosystem. By fusing geospatial data, satellite imagery, and advanced AI models, we empower farmers, financial institutions, and government bodies with actionable insights to transform decision-making from "guesswork" to "precision."

---

## 🚀 Key Features

- **🧠 AI Yield Predictor:** Forecast regional crop yields with high accuracy using multi-layered data (Satellite + Soil + Weather).
- **🛰️ Satellite Crop Health:** Real-time monitoring of crop stress and growth stages using remote sensing.
- **📈 Market Intelligence:** Predictive analysis for Mandi prices to optimize harvest-to-market timing.
- **🏦 Loan Risk Engine:** Data-backed credit scoring for farmers, enabling faster and more equitable loan disbursal for banks.
- **🔗 Blockchain Transparency:** Immutable records for land ownership and supply chain traceability to ensure trust.
- **⚡ Hyper-local Weather Alerts:** Proactive notifications to safeguard crops from extreme weather events.

## 📖 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- Firebase Account

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/krishi-mitra-ai.git
   cd krishi-mitra-ai
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root and add your Firebase and Gemini API keys:
   ```env
   GOOGLE_GENAI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
